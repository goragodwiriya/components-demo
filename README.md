# Components Demo

โปรเจ็คตัวอย่างนี้เป็นเว็บแบบสแตติก (static) ขนาดเล็กที่แสดงการจัดเรียงส่วนประกอบ HTML/CSS/JS แบบง่ายเพื่อใช้เป็นตัวอย่างหรือทดสอบการพัฒนาหน้าเว็บอย่างรวดเร็ว

## ภาพรวม

- โฟลเดอร์โปรเจ็ค: root
- เป้าหมาย: แสดงตัวอย่างการใช้งานไฟล์ HTML, CSS และ JavaScript แบบเรียบง่าย โดยมีไฟล์ตัวอย่างสำหรับส่วนประกอบ (`components.html`) และหน้าเริ่มต้น (`index.html`).

## โครงสร้างไฟล์

- `index.html` - หน้าเริ่มต้นของเว็บ (entry point)
- `components.html` - หน้าตัวอย่างที่รวบรวม component / ส่วนประกอบต่าง ๆ
- `styles.css` - ไฟล์สไตล์หลัก (CSS)
- `script.js` - สคริปต์ JavaScript สำหรับฟังก์ชันหรือการโต้ตอบแบบง่าย

## วิธีใช้งาน (Run / Preview)

เนื่องจากเป็นเว็บสแตติก คุณมีตัวเลือกดังนี้:

1) เปิดไฟล์ `index.html` โดยตรงในเบราว์เซอร์ (เหมาะสำหรับทดสอบแบบพื้นฐาน)

2) รันเว็บเซิร์ฟเวอร์แบบง่าย (แนะนำเมื่อต้องการเรียกไฟล์ด้วย fetch หรือเส้นทางที่ต้องการ HTTP):

```bash
# Python 3 (พอร์ต 8000)
python3 -m http.server 8000

# แล้วเปิดเบราว์เซอร์ที่
http://localhost:8000/
```

หรือถ้าคุณใช้ Node.js:

```bash
# ติดตั้ง http-server (ถ้ามี npm)
npm install -g http-server
# รัน
http-server -p 8000
# เปิด
http://localhost:8000/
```
